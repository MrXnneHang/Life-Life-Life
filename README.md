在这个仓库诞生这么久，我才第一次想到我大概能为它写什么。

这个仓库本来是我的一些日常记录，我个人颇爱动笔，这句话可能存在纰漏，最早也确实是用笔，但后来变成了码字。两者本质也许并无不同，但我不免有些怀念以前真正用笔的时候。两者思考的节奏是完全不同的。一个是思考费力在后面追，一个是缓缓而行。

## 手记 - 水云身

我能记忆的最早的我有计划的记录，是对于影视，小说，游戏等虚构作品人物的 To somebody。

比如写给看完言叶之庭写下给雪野，看完伪恋写下给金毛毛（我实在不记得女主名字了，即便当时刚刚看完），三日间的幸福后的给宫城等等，之所以写这些是因为初高中时用于看完作品后记录当时的一些感觉来排解寂寥和空虚之感。可能也是这个习惯让我逐渐成了 INFP。

这些仍然在我的纸质日记本上。如果对应现在的应当就是**手记**的版块。例如:

[如花束般的恋爱 | 文青是种病，开始亦是结束，深度共鸣不一定需要表面合拍。](https://xnnehang.top/notes/54)

 我写观后感的形式经过了信件，评论各种形式的多次迭代，有的时候让我觉得很“装”，但事后看上去还不错。但我最喜欢的仍然是写给角色的，因为回看的时候会有种和以前的自己对话的感觉，但又有点幼稚。这种范式通常不是刻意的，也许正应了那句“情不知所起，一往而深”（或许我拽文不当了请原谅，但有时在看完一部作品后确实会有种忍不住想冲进去作品和人物面对面对话的冲动），例如这是我最近的一篇日记式的观后：

[《你是我的荣耀》观后。](https://xnnehang.top/notes/39)

我显然做不到冲进作品，所以我写下了那些信，也许是告别信，但更早我记得写了不少告白信 =-=，例如我曾给宫城告过三次白，有长有短。但那段时间正好是我阅读经验的成长期，我把我最早的热情都给了《三日间的幸福》和《挪威的森林》。

在高中时期，我把这个内容看作我能够记录的全部，当时的我满心诗和远方，虽然我对唐诗宋词不过半吊子的喜欢，一种只是流于直觉和冲动的喜欢，而不会深入理解。

如果要给这个板块命名，我希望它叫水云身。我希望我在记录它的时间里，我的身体和灵魂如同云水般束缚不住，也希望能够在记录时达到那种周身不扰的境界。

而我最近常常处于困厄之中，当罚，罚我看一遍以前的手记，以及把一些纸质日记上的搬运过来。

最近我把自己搞得很难受，常常觉得自己只有一个人，又觉得自己浪费过太多时间，很多东西还没体验或者实现就来不及了，太多时间不曾动笔记录，而断更越久，继续更新的勇气便越少，最后连再次打开都不敢。

但是，我曾深深地沉入那些作品里，那段时间不曾有假，即便占比极少，那些角色也确实影响过我，我并不是一个人，当然这句话听上去好像日系凡尔赛，或者说主角的嘴炮：“**君は一人じゃない！**”

但是我在认清那点的一瞬间，我确实有种解脱感。

我该感谢他们，以及，之后手记的路我希望它依然十分浪漫。

在《你是我的荣耀》里，关在给一号命名搜神号时说出了他们的初衷：

> 在茫茫宇宙中，寻找地球和人类起源的真相。在无穷无尽中，寻找另一神迹，证明人类并不孤单。

而我，不求那么高大上的，我只求我依然能够在我写手记的时间里做着白日梦，梦那虚幻的长安盛世。它是属于故事的，是属于浪漫的。

> 很多东西还没体验或者实现就来不及了

对于这点我倒是有解，我从来不奢望做一个现充，在高中时我通过看小说来填补，而我想我现在应该也能找到法子。

最后我也希望我不要忘了，写字，或者码字，应该是一种如此安心的感觉。

虽然。。现在讲这话很不合时宜，我写的这些东西大概会在以后成为没有意义的互联网垃圾，或许已经是了，但是它对我有意义，而只要有任何一个人产生过同感，我的记录便更值得。有幸被催更过几次，但我都避过了。也许该考虑怎么复健了。

当然，在那之前，玉兔学神救我狗命啊。


## 博文 - 出仕心

这一块，我就简单而言，不再像上面那边扯得很远。

- [ ] 我非常纠结于自己分类归档，以及打标签。那么标签干脆让 AI 来打，分类只需要分出几个大类。不需要太详细。
- [ ] 我的博客并没有提供关键词检索和标题检索功能，而且对于这个博客系统我也不是很熟悉，修改起来不优雅而且还费力，可以留一个地图，来引导检索，也可以作为二次界定。
- [ ] 我纠结于过于依赖大模型的 debug 记录不过是在给 LLM 打下手，非常枯燥，而且大模型不一定是对的，在下一些结论的时候会缺少依据，空口无凭。 也许可以分成两大块，根据依赖大模型的程度来分（复制粘贴一级，理解二级，查证三级，自己写四级。引用独立），另外在文章开头给出一些界定符，比如二级以下该文章依赖 AI 生成，可信度有待商榷之类的。

关于实际的内容。 光写 debug 实在无趣，而写教程也陷入像廖雪峰博客里提到的把知道的东西写下来是无聊的，探索才是让人动力迸发的，一边探索一边写，或许正印了费曼学习法。不过费曼学习法得不断有人更深入地提出问题，这点可以让 AI 来，或者留待有缘人。

另外我更希望自己可以像 [科学空间](https://spaces.ac.cn/) 那样子，写所有自己感兴趣的，哪怕那个领域只是门外汉，也许因此去学才更有意思。

我最近一直困扰的一点是，随着越发临近毕业，我感到自己之前那种学习欲越发消退，只是想要遁入手记的虚幻世界，让我怀疑起自己是否从未发自内心的去学东西，而只是为了嘉奖，虚荣心，工作，或者其他别的东西。

我努力回想自己是否有曾经发自内心地想要学某个东西，期间我理解了为什么大人总是对小孩子说小时候要努力学习。因为我发现我真正学习的时间百不存一，会不自觉地想如果时光倒流会如何？这种在小说里才会出现的桥段我考虑起来相当顺畅，就好像我曾一次次考虑过的这样一个问题:

> **而作为人类，作为死神口中的大部分普通人之一，没有碰到美少女拯救的我，总是会在假期开始时感到喜悦，前期感到漫无目的，中期感到无聊，后期寻求刺激。然后在末期，感觉到似乎得努力把之前缺失的部分都补回来的心情而开始规划作息。如此一次次“死去”，直到最终死去。**

———— [《没有明天的我们，在昨天相恋》 后 -在寒假结束时“死去”。](https://xnnehang.top/notes/8)

这种很矫情的想法，在后来我发现和 LLM 的 Lost in the middle 现象极度吻合，另外它似乎还是人类的注意力底层机制，叫做序列位置效应（serial position effect）。

这个发现让我觉得有意思，但绝不是我的借口，说起来关于序列位置效应那篇我还没写完就鸽了。

从小到大，我从来没有一次真正地完完整整地”过好了“寒暑假。而这何尝不是一次次的模拟人生，尽管它是故事模式的。

中间我写过一些类似于 [《存在主义心理治疗》 阅读手记 - “我必须死，才能好好活着”](https://xnnehang.top/notes/1) 这样的东西来让自己获得暂时性解脱。但我知道这些由头在我块盖棺定论的时候压根没啥作用。

也许类似存在主义心理治疗这样的文章应该属于博文而非手记？我的分类焦虑又出现了。

暂时先不想那么多，我只需要考虑写。至于它究竟属于什么，写出来再说。
